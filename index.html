pokebased

[ mc.pokebased.net ]

About

pokebased is a Minecraft server built around the Cobblemon mod. It also features the Open Parties and Claims mod for grief protection. OPC is not required to connect to the server. However, it and several other mods from Xaero make it more convenient to use.

Chat reporting is disabled on this server.

Rules

  • No hacking.
  • No spamming.
  • No bug abuse.

More rules will be added if people abuse our leniency.

Mods

Required:

  • Fabric 1.19.2
  • Cobblemon (requires Fabric API and Architectury API)

Recommended:

  • Open Parties and Claims (requires ForgeConfigAPIPort)
  • Xaero's Minimap
  • Xaero's World Map

If you install Xaero's World Map, you will need to change the keybind for either Xaero's World Map's "Open World Map" control or Cobblemon's "Open Summary" control in your in-game settings as both are set to the 'm' key by default.

You can install Sodium, Lithium, and/or Phosphor to improve performance, if you would like to. You can also install No Chat Reports if you would like to verify that chat reporting is disabled.
